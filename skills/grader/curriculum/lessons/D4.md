# Output Format & Structure

Describing the shape you want is weaker than showing it. A tiny skeleton, JSON stub, or section list removes ambiguity faster than adjectives.

Use delimiters such as triple quotes, XML tags, or markdown headers to separate instructions from data. That separation helps the model treat the data as content, not as a command.
