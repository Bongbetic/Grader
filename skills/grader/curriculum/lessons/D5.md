# Model & Parameter Fit

The same prompt can be excellent for one model class and harmful for another. Model fit is the decision to tune the prompt to the actual model that will run it.

Standard models usually benefit from clearer structure and, on hard tasks, explicit reasoning steps. Reasoning models prefer lean, direct prompts and native reasoning controls rather than forced chain-of-thought. See `model-fit-meta` for the full divergence map.
