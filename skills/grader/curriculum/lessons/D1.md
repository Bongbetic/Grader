# Objective & Success Criteria

A prompt that names a task but not "done" leaves the model guessing. The objective is the action verb; the success criteria tell you how to check the result.

Start every prompt with a clear action and one checkable sentence. If you cannot write a pass/fail test in one line, the model cannot know what you want either.

- Try this: Rewrite "help with marketing" as "Write a 120-word launch email. Success: reader knows the main benefit and CTA in one read."
