# Examples / Shot Design

Examples are a powerful way to teach format, but they are not free. One aligned example beats three inconsistent ones. The model pays close attention to details you may not intend.

Start with zero or one example. Add a second only if the output still drifts. Keep format, tone, and length identical across examples. For reasoning models, prefer zero-shot or at most one example.

- Try this: Add one example of the exact output shape you want, then remove it and compare. Keep whichever wins.
